# React

Package: [`@santi020k/eslint-config-react`](https://www.npmjs.com/package/@santi020k/eslint-config-react)

```js
import { eslintConfig } from '@santi020k/eslint-config-basic'
import react from '@santi020k/eslint-config-react'

export default eslintConfig({
  typescript: true,
  frameworks: {
    react
  }
})
```

Use this package for React and Hooks rules in plain React apps or as the explicit React dependency for Next.js and Expo.
